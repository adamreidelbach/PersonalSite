Get Git

Get Git is an application built in AngularJS that helps new developers effectively and efficiently learn to use git. Users are tasked with completing common git scenarios developers encounter on a daily basis. Each section provides step by step instructions, a virtual terminal to input commands, and a visual representation of commands as they are entered.

Technologies: AngularJS, JavaScript, JQuery, Materialize, Angular-Materialize and Sass.
